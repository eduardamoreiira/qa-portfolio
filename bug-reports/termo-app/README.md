# 📱 Bug Reports — Termo App (iOS)

Análises exploratórias realizadas no aplicativo **Termo**, jogo de palavras para dispositivos móveis disponível na App Store. O app é a versão brasileira do Wordle, desenvolvido para o mercado de língua portuguesa.

> ⚠️ **Aviso:** Os bugs documentados aqui foram encontrados durante análise exploratória independente, sem qualquer vínculo com os desenvolvedores do aplicativo. O objetivo é exclusivamente demonstrar capacidade técnica de identificação e documentação de defeitos.
