# 👩‍💻 Portfólio de QA — Eduarda Moreira

Oi! Sou apaixonada por qualidade de software e acredito que um bom QA vai muito além de 'achar bugs' — trata-se de construir confiança no produto. Atualmente, estou mergulhada no universo de testes manuais e automação, documentando cada descoberta.

Criei este espaço para registrar minha jornada técnica. Os projetos aqui são análises de sistemas reais (mobile e web), onde exercito meu olhar crítico e organizo processos de QA que facilitam a vida de qualquer time de desenvolvimento.
